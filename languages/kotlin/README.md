# feeling-rusty Kotlin bindings

Kotlin bindings for feeling-rusty, generated by [UniFFI](https://mozilla.github.io/uniffi-rs/).

CI produces a **single multi-platform JAR**: one artifact containing the Kotlin bindings and native libraries for all supported platforms (darwin x86_64/arm64, linux x86_64/arm64, windows x86_64/arm64). JNA loads the correct native lib for the current OS/arch from the JAR at runtime.

For local development, run `uniffi-bindgen generate --library <path-to-lib> --language kotlin --out-dir out` and copy the contents of `out/` into `src/main/kotlin/`, then copy the native library for your platform into `src/main/resources/` under the JNA resource path (e.g. `linux-x86-64/`, `darwin-aarch64/`).

## Build

```bash
./gradlew build
```

## Requirements

- JNA 5.12.0+
- Kotlin 1.6+ (for coroutines)
